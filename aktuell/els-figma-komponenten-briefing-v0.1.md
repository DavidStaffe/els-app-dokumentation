# ELS – Figma-Komponenten-Briefing V0.1

Stand: April 2026

## Design Foundations
- Farbsystem: Primaer, Sekundaer, Status (rot/gelb/gruen/grau), Hintergrund
- Typografie: 3 Groessen (Heading/Body/Caption), 2 Gewichte
- Spacing: 4px-Grid
- Iconset: einheitlich (Lucide oder HeroIcons)

## Core-Komponenten
- App Shell (Navigation, Sidebar, Header)
- Global Header (Incident-Kontext, Rolle, Status)
- KPI-Kachel (Zahl, Label, Trend-optional)
- Status-Badge (farbkodiert je Status)
- Filterleiste (Chips/Dropdowns)
- Listen-Container (Tabelle/Karten, sortierbar)
- Quickview-Panel (Seitenleiste oder Modal)
- Formulargruppen (Label, Input, Validierung)
- Primaer-Aktionsleiste (Hauptbutton, Sekundaerbutton)

## Domain-Komponenten
- Patientenkarte (Kennung, Sichtung, Verbleib, Status)
- Transportkarte (ID, Typ, Ressource, Ziel, Status)
- Ressourcenkarte (Typ, Status, Zuordnung)
- Konfliktkarte (Typ, Schwere, Aktion)
- Workflow-Aktionskachel (Schritt, Status, CTA)
- Statusmatrix-Ressource (Typ x Status-Grid)

## Layoutmuster
- Zweispalten (60/40 oder 50/50)
- Listenansicht mit Detailpanel
- Vollbild-Bericht (A4, Sidebar-Nav)
- Abschluss-Check (Ampel-Header + Tab-Bereich)

## Figma-Seitenstruktur (empfohlen)
1. Foundations
2. Core Components
3. Domain Components
4. Screens / Flows
5. Archiv
